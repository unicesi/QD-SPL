-----------------------------------------------------------------------------
1. INSTALLING oAW 4.3.1 PLUG-IN MANUALLY
-----------------------------------------------------------------------------

Locate the oAW 4.3.1 zip file in the lib folder. To install the plug-in,
unzip the file into the Eclipse installation directory (or the plugins folder
depending on how the plug-in is packaged). The plug-in will not appear until
Eclipse is restarted.
